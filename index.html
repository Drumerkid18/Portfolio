Whack-A-Mole

A game of speed and focus!

I was given the assignment to create a game where an image would appear and disappear in a matter of moments whether the user clicked on it or not. For this I used jQuery, CSS, PHP, HTML, Javascript. I kinda went a little overboard and had a total of 36 moles, and they fired off at random times, like the real game, it wasn't predictable or easy at all. I also randomized the images that would display for added fun. I created a start page and each game lasted 1 minute. At the end it would tally your score. I had a lot of fun with this project and added a lot of personality to it.

  • Date: January 2015
  • Client: Codeup Bootcamp
  • Category: Jquery Practice, and everything else we had already learned up to this point.

National Parks

A list of some of the Nations Parks

This was a fun project, it taught us how to use a database. A huge component in everyday websites. On this project we seeded data, and also uploaded new data using a form. The number of pages displayed is directly related to the number of parks in the database. Also the pages showing depends on which page you are on.

Address Book

Place for storing addresses!

This project took a little longer, because it covered a lot of different things. We implemented adding a contact, an address, and phone number. We have all the states pre-saved to avoid any strange input by the user. You can add a new contact, edit the contact, delete the contact, add 1 or more addresses to that contact and delete those as well.

  • Date: January 2015
  • Client: Codeup Bootcamp
  • Category:MySQL practice, jQuery events, CSS styling, PHP, and everything else we had covered.

Insure Joy

My biggest Project

This was the final project we completed as a team. It was a Web Application for purchasing Life Insurance. Built using Laravel, PHP, MySQL, JavaScript, jQuery, Twitter Bootstrap, Dropzone.js. Version control using Git and GitHub. Features user logins, creating user, saving quotes, saving applications, admin/company dashboard, profile/user dashboard, 3rd party integration, upload files, forgot password, contact messaging. This could very easily become a business or used for a broker who wants online presence.
